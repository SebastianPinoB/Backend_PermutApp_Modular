# PRODUCTO

Contenido del producto backend solicitado para la entrega academica.

- `Codigo fuente/`: microservicios Spring Boot de PermutApp.
- `Scripts SQL/`: pendiente de incorporar scripts finales de base de datos si el equipo los exporta desde Supabase.
- `Datos de prueba/`: pendiente de incorporar datos finales de prueba si el equipo los define para la entrega.
- `Librerias y configuracion/`: la configuracion principal esta dentro de cada servicio (`pom.xml`, `.env.example`, `src/main/resources/application.properties`).
